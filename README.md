Recreating the Google homepage using only HTML & CSS.


## Built With

* HTML
* CSS

## Concepts Used
* Setting up a productive development environment with VS code and Git.
* HTML page structuring and semantic tags.
* CSS styling & positioning.
* Flexbox.
